각 서류별 출력 방향 유지

원래 요청 범위 반영표

비품관리 항목은 삭제됨
1. 시작 화면 달력
  • 월 선택 시 해당 월 달력 표시
  • 공휴일, 토요일, 일요일 자동 표시
  • 각 월 생성 내용을 낱장 출력
2. 주요 탭
  • 근무표
  • 업무 분장
  • 월말 결재 서류
  • 체크리스트
3. 월말 결재 서류
  • 냉장고 온습도 관리 기록부
  • 의료기기 일상 점검표
  • 조제장비관리기록지
  • 장비명과 점검 항목 편집 가능
4. 체크리스트
  • 외용제 단독 가로 출력
  • 추/긴 업무 + ATC 업무 가로 출력
  • PTP 업무 + 주사 업무 가로 출력
  • 노트북 1~4번 세로 출력
5. 직원 업무 분장
  • 월별 이름 순환 배정
  • B~D 이름 수기 편집
  • 흰색 셀 이름과 점심 시간 이동 규칙 반영
6. 약제팀 업무 분장
  • 선택 월 자동 반영
  • A행 이름 편집
  • 지정 약사별 업무 수기 편집
7. 약사 근무표 자동 배정
  • 나이트 약사 6주 턴 규칙
  • 나이트 직원 3일 근무/3일 휴무 규칙
  • 토요일, 일요일, 공휴일 직원/약사 배정
  • 일정 목록 입력 시 달력 표시
8. 개발 산출물
  • TypeScript 앱
  • AGENTS.md 100줄 미만
  • 시스템 맵핑 문서
  • GitHub 저장 및 자동 푸시 흐름

진행 상태

프롬프트 정리원본을 마크다운으로 변환하고, 잘못 섞인 비품관리 문구를 삭제했습니다.
자료 확인근무표, 업무분장, 월말 서류, 체크리스트 엑셀 구조와 출력 방향을 확인했습니다.
화면 설계현재 화면은 실제 앱이 아니라 설계 목업입니다. 누락 범위를 줄이기 위해 요청 범위 반영표를 추가했습니다.
