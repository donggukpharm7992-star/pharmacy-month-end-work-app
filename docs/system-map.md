# 약제팀 월말 업무 앱 시스템 맵

## 목적

이 문서는 이후 코딩 에이전트가 앱 구조와 업무 규칙을 착각하지 않도록 돕는 기준 문서다.

## 원본 파일 매핑

| 원본 파일 | 앱 영역 | 처리 방식 |
| --- | --- | --- |
| `약제팀 근무표.xlsx` | 근무표 | 월별 근무표 레이아웃과 8월 기준 규칙 참고 |
| `직원업무분장_2026.xlsx` | 업무 분장 | 직원 업무명, 시간대, 월별 순환 구조 참고 |
| `약제팀 업무분장_2026.xlsx` | 업무 분장 | 약제팀 업무표와 지정 약사 업무 참고 |
| `★냉장고 온습도 관리기록부20180911.xls` | 월말 결재 서류 | 하위 탭 및 편집 가능한 점검 항목 |
| `★의료기기일상점검표 (조제실)190902.xlsx` | 월말 결재 서류 | 자산번호, 장비명, 날짜별 점검칸 |
| `★조제장비관리기록지(산제,정제,청소일지).xls` | 월말 결재 서류 | 산제, 정제, 청소일지 출력 |
| `직원 업무 체크 리스트.xlsx` | 체크리스트 | 직원 업무 체크리스트 출력 그룹 |

## 주요 모듈

| 모듈 | 책임 |
| --- | --- |
| `src/domain/calendar.ts` | 날짜 키, 월 달력, 주말, 2026 공휴일, 일정 배지 |
| `src/domain/schedule.ts` | 나이트 약사, 나이트 직원, 주말/공휴일 배정, 주 단위 근무표 묶음 |
| `src/domain/taskRotation.ts` | `직원업무분장_2026.xlsx` 기반 직원 업무 분장 열과 월별 이름 순환 |
| `src/domain/documents.ts` | 월말 서류와 체크리스트 출력 묶음 |
| `src/domain/assignmentViews.ts` | 업무 분장 하위 탭과 출력 단위 |
| `src/domain/pharmacistAssignment.ts` | `약제팀 업무분장_2026.xlsx` 기반 약사 업무 분장 틀과 편집 가능 칸 |
| `src/storage.ts` | localStorage 기반 편집값 저장 |
| `src/App.tsx` | 탭, 월 선택, 편집 UI, 출력 버튼 |

## 자동 배정 규칙

- 나이트 약사 2026년 8월 기본 순서: 윤주원, 정순미, 송유희, 김동신, 이상훈, 장소희.
- 2026-08-10부터 6주마다 앞에서 네 번째 이름을 맨 뒤로 보내며, 해당 턴 변경일은 달력과 근무표 일정 행에 자동 표시한다.
- 2026-08-01은 4번 김동신+1번 윤주원 조합으로 시작한다.
- 나이트 약사는 직원처럼 3일 연속 근무하지 않는다. 기본 리스트의 1번+4번, 2번+5번, 3번+6번, 4번+1번, 5번+2번, 6번+3번의 6개 순서 조합을 날짜별로 순환한다.
- 근무표 기본 일정은 공휴일을 제외한 평일 기준으로 계산한다. 첫 번째 목요일은 유효기간조사, 두 번째 화요일은 월례회의, 두 번째 목요일은 대청소, 세 번째 수요일은 경구 재고조사, 세 번째 목요일은 주사 재고조사, 네 번째 수요일은 직원업무 변경일이다.
- 근무표 기본 일정 날짜는 선택 월별로 표시하며, 하단 날짜 입력에서 변경한 값은 해당 월에 저장하여 근무표와 달력에 즉시 반영한다.
- 근무표 상단에는 연월과 `휴가자 연차 2명` 부제를 표시하며 부제는 편집할 수 있다. 월간 달력 인쇄물에도 선택 연월을 표시한다.
- 나이트 직원은 세 포지션을 3일 근무/3일 휴무 규칙으로 교대하며, 2026년 8월 근무표 흐름을 이어 9월부터 포지션별 시작일을 다르게 둔다.
- 토요일 `07:15-11:15`에는 직원 2명을 배정한다.
- 일요일과 공휴일 `08:00-12:00 아래`에는 직원 1명을 배정한다.
- 주말 직원 기본 순서는 김동희, 박종연, 김지은, 김지현, 박지숙, 송현우, 김서훈, 심관석이다. 토요일 2명과 일요일 1명은 하나의 순번을 공유하며 2026-09-05 박종연+김지은부터 시작한다.
- 공휴일 직원은 2026-08-17 김지현 고정이며, 2026-09-24 김서훈부터 별도 순번으로 순환한다. 2026년 8월의 나머지 직원 배정은 기존 근무표를 유지한다.
- 근무표 하단 직원 목록은 `번호+이름`으로 편집하며, 목록 끝 또는 각 이름 다음에 새 이름을 추가하거나 기존 이름을 삭제할 수 있다. 변경된 이름과 순번은 주말·공휴일 배정에 즉시 반영한다.
- 선택 월에 나이트 턴 변경일이 있으면 하단 입력란에 해당 변경일을 표시한다. 날짜를 수정하면 6주 주기의 기준일도 같은 일수만큼 이동한다.
- 근무표의 자동 배정 이름은 셀에서 직접 수정할 수 있으며, 수기 수정값은 localStorage에 저장되어 자동 계산값보다 우선한다.
- 근무표는 상단 연월과 편집 가능한 휴가자 부제를 포함하여 A4 가로 한 장으로 출력한다.
- 매월 첫 번째 토요일 `08:00-17:00`은 최윤영+이승현 고정이며, 두 번째 토요일부터 두 번째 약사는 이승현으로 고정하고 첫 번째 자리에 약사 목록 순번을 배정한다.
- 서윤석은 일요일에만 월별 위치를 번갈아 고정 배정하고 남은 자리에 약사 목록 순번을 배정한다. 공휴일에는 서윤석 고정을 적용하지 않고 약사 목록 순번대로 2명을 배정한다.
- 일정 목록 날짜는 시작 달력 배지와 약제팀 근무표의 주간 일정 행에 표시한다.
- 약제팀 근무표의 토요일·일요일 이름 입력 영역은 유색 셀로 표시한다.
- 직원 업무 분장의 `시 간` 이름은 2026년 8월을 기준으로 추/긴부터 송현우, 심관석, 김지은, 박지숙, 김서훈, 김동희, 김지현, 박종연 순서이며, 매월 각 이름이 다음 업무로 한 칸씩 내려간다.
- `7:15~8:00` 이름은 2026년 8월 추/긴, 외용, PTP반납1에 박지숙, 김지현, 김동희 순서로 배정하며, 매월 한 칸씩 순환한다. `7:15~8:00`과 `8:00~11:30` 사이에는 별도 빈 열을 두지 않는다.
- 직원 업무 분장의 하단 `시 간` 및 `7:15~8:00` 이름 목록 순서를 편집하면 업무 배정 순서도 즉시 다시 계산하며 localStorage에 저장한다.
- 직원 업무 분장의 식사 기본값은 `시 간` 이름이 `7:15~8:00` 이름 목록에 포함되면 `11:45-12:30` 쪽 `식사`, 나머지는 `12:30-13:30` 쪽 `식사`다. 표 안의 식사 칸을 수기로 수정한 값은 localStorage 값이 우선한다.

## 출력 단위

- 시작 달력: 선택 월 단위 낱장 출력.
- 근무표: `약제팀 근무표` 제목을 사용하고, 원본 엑셀처럼 `근무시간 | 월 | 화 | 수 | 목 | 금 | 토 | 일` 주 단위 블록으로 출력.
- 업무 분장: 직원 업무 분장과 약사 업무 분장을 하위 탭으로 분리하고, 선택한 표를 Excel에서 열 수 있는 `.xls` 파일로 출력한다.
- 직원 업무 분장: `직원업무분장_2026.xlsx`의 오전/오후 업무 원문과 `대청고`, `재고 조사/유효기간`, `약 채우기` 열을 모두 표시하고 모든 칸은 수기 편집 가능.
- 약사 업무 분장: `약제팀 업무분장_2026.xlsx`의 표 틀을 기준으로 하며 모든 보이는 칸은 수기 편집 가능.
- 약사 업무 분장의 이름 행은 위치를 고정한다. 하단 약사 이름 목록은 표시명만 바꾸며, 고정 그룹은 업무를 유지하고 순환 그룹은 이름 위치를 유지한 채 업무 내용만 매월 순환한다. 박현영은 오후 업무 칸을 고정한다.
- 월말 결재 서류: 각 하위 문서별 출력. 하위 문서 안에서는 원본 엑셀 시트 목록을 하단 `출력 항목 목록`으로 보여주고, 선택한 항목만 본문에 표시한다.
- 냉장고 온습도 관리 기록부 출력 항목: 병동약국-오전, 병동약국-오후, 제제실-오전, 제제실-오후, 온습도, 외래약국.
- 의료기기 일상 점검표 출력 항목: 조제실. 장비 목록은 하단에 별도 버튼으로 제공하고, 클릭한 장비 행을 본문에서 강조한다.
- 조제장비관리기록지 출력 항목: 산제, 정제, 청소일지.
- 직원 체크리스트: 외용제 단독, 추/긴+ATC, PTP+주사. 각 표는 선택 월의 1일부터 말일까지 날짜/요일을 가로로 표시하고 주말·공휴일은 유색 처리한다.
- 노트북 체크리스트: 1번부터 4번까지 세로 월간 표로 출력하며, 선택 단추와 전체 출력 모드를 제공한다.

## 제외 항목

- 비품관리 화면은 다른 앱 프롬프트가 섞인 내용이므로 구현하지 않는다.
