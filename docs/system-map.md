# 약제팀 월말 업무 앱 시스템 맵

## 목적

이 문서는 이후 코딩 에이전트가 앱 구조와 업무 규칙을 착각하지 않도록 돕는 기준 문서다.

## 원본 파일 매핑

| 원본 파일 | 앱 영역 | 처리 방식 |
| --- | --- | --- |
| `약제팀 근무표.xlsx` | 근무표 | 월별 근무표 레이아웃과 8월 기준 규칙 참고 |
| `직원업무분장_2026.xlsx` | 업무 분장 | 직원 업무명, 시간대, 월별 순환 구조 참고 |
| `약제팀 업무분장_2026.xlsx` | 업무 분장 | 약제팀 업무표와 지정 약사 업무 참고 |
| `★냉장고 온습도 관리기록부20180911.xls` | 월말 결재 서류 | 하위 탭 및 편집 가능한 점검 항목 |
| `★의료기기일상점검표 (조제실)190902.xlsx` | 월말 결재 서류 | 자산번호, 장비명, 날짜별 점검칸 |
| `★조제장비관리기록지(산제,정제,청소일지).xls` | 월말 결재 서류 | 산제, 정제, 청소일지 출력 |
| `직원 업무 체크 리스트.xlsx` | 체크리스트 | 직원 업무 체크리스트 출력 그룹 |

## 주요 모듈

| 모듈 | 책임 |
| --- | --- |
| `src/domain/calendar.ts` | 날짜 키, 월 달력, 주말, 2026 공휴일, 일정 배지 |
| `src/domain/schedule.ts` | 나이트 약사, 나이트 직원, 주말/공휴일 배정, 주 단위 근무표 묶음 |
| `src/domain/taskRotation.ts` | `직원업무분장_2026.xlsx` 기반 직원 업무 분장 열과 월별 이름 순환 |
| `src/domain/documents.ts` | 월말 서류와 체크리스트 출력 묶음 |
| `src/domain/assignmentViews.ts` | 업무 분장 하위 탭과 출력 단위 |
| `src/domain/pharmacistAssignment.ts` | `약제팀 업무분장_2026.xlsx` 기반 약사 업무 분장 틀과 편집 가능 칸 |
| `src/storage.ts` | localStorage 기반 편집값 저장 |
| `src/App.tsx` | 탭, 월 선택, 편집 UI, 출력 버튼 |

## 자동 배정 규칙

- 나이트 약사 기본 순서: 윤주원, 정순미, 송유희, 이상훈, 장소희, 김동신.
- 2026-09-21부터 6주마다 앞에서 네 번째 이름을 맨 뒤로 보낸다.
- 나이트 약사는 직원처럼 3일 연속 근무하지 않는다. 기본 리스트의 1번+4번, 2번+5번, 3번+6번을 짝으로 하여 날짜별로 순환한다.
- 나이트 직원은 세 포지션을 3일 근무/3일 휴무 규칙으로 교대하며, 2026년 8월 근무표 흐름을 이어 9월부터 포지션별 시작일을 다르게 둔다.
- 토요일 `07:15-11:15`에는 직원 2명을 배정한다.
- 일요일과 공휴일 `08:00-12:00 아래`에는 직원 1명을 배정한다.
- 토요일 `08:00-17:00` 두 번째 약사는 이승현 고정이다.
- 서윤석은 일요일에만 월별 위치를 번갈아 고정 배정하며, 공휴일에는 서윤석 고정을 적용하지 않고 약사 리스트 순번대로 2명을 배정한다.
- 일정 목록 날짜는 시작 달력 배지와 약제팀 근무표의 주간 일정 행에 표시한다.
- 약제팀 근무표의 토요일·일요일 이름 입력 영역은 유색 셀로 표시한다.
- 직원 업무 분장에서 김동희, 박지숙, 지현이 `시 간` 칸에 들어가면 점심은 `11:45-12:30`으로 표시한다.

## 출력 단위

- 시작 달력: 선택 월 단위 낱장 출력.
- 근무표: `약제팀 근무표` 제목을 사용하고, 원본 엑셀처럼 `근무시간 | 월 | 화 | 수 | 목 | 금 | 토 | 일` 주 단위 블록으로 출력.
- 업무 분장: 직원 업무 분장과 약사 업무 분장을 하위 탭으로 분리해 각각 한 장씩 출력.
- 직원 업무 분장: `직원업무분장_2026.xlsx`의 오전/오후 업무 원문과 `대청고`, `재고 조사/유효기간`, `약 채우기` 열을 모두 표시하고 B~D 이름 칸은 수기 편집 가능.
- 약사 업무 분장: `약제팀 업무분장_2026.xlsx`의 표 틀을 기준으로 하며, 이름 칸 전체와 이지은/박혜정/김경원/김수빈/박주영 업무 칸, 박현영 오전 업무 칸만 수기 편집 가능.
- 월말 결재 서류: 각 하위 문서별 출력.
- 직원 체크리스트: 외용제 단독, 추/긴+ATC, PTP+주사.
- 노트북 체크리스트: 1번부터 4번까지 세로 출력.

## 제외 항목

- 비품관리 화면은 다른 앱 프롬프트가 섞인 내용이므로 구현하지 않는다.

현재 저장소는 로컬 git을 기준으로 준비한다. 원격 URL이 생기면 `git remote add origin <URL>` 후 `scripts/git-auto-push.ps1`을 사용한다.
