# 약제팀 월말 업무 앱 설계

## 목표

폴더의 기존 엑셀 문서를 기준으로 근무표, 업무 분장, 월말 결재 서류, 체크리스트를 한 화면에서 월별로 생성·수정·출력하는 TypeScript 앱을 만든다.

## 범위

- 시작 화면 달력은 선택 월의 날짜, 토요일, 일요일, 2026년 관공서 공휴일, 수기 입력 일정을 표시한다.
- 주요 탭은 `근무표`, `업무 분장`, `월말 결재 서류`, `체크리스트` 네 개다.
- 월말 결재 서류는 `냉장고 온습도 관리 기록부`, `의료기기 일상 점검표`, `조제장비관리기록지` 하위 탭으로 나누고 출력할 수 있다.
- 체크리스트는 직원 업무 체크리스트와 노트북 체크리스트로 나눈다.
- 직원 업무 체크리스트 출력은 외용제 단독, 추/긴+ATC, PTP+주사 세 묶음의 가로 출력이다.
- 노트북 체크리스트는 1번부터 4번까지 세로 출력이다.
- 직원 업무 분장과 약제팀 업무 분장은 월 선택값을 반영하고 이름/업무 수기 편집을 허용한다.
- 약사 근무표는 나이트 약사 6주 턴, 나이트 직원 3일 교대, 주말/공휴일 배정 규칙을 반영한다.
- 비품관리 화면은 잘못 섞인 다른 앱 요구사항이므로 제외한다.

## 구조

앱은 Vite + React + TypeScript로 구성한다. 업무 규칙은 `src/domain`의 순수 함수로 두고, 화면은 `src/App.tsx`와 CSS에서 조립한다. 편집 상태는 브라우저 localStorage에 저장하여 별도 서버 없이도 사용할 수 있게 한다.

## 문서와 에이전트 안내

`AGENTS.md`는 100줄 미만으로 유지한다. `docs/system-map.md`는 탭, 데이터 원천, 자동 배정 규칙, 출력 그룹, 수정 시 주의사항을 담아 이후 코딩 에이전트가 혼동하지 않게 한다.

## GitHub

로컬 git 저장소를 만들고 전체 폴더를 커밋한다. GitHub 원격 주소나 인증 가능한 도구가 제공되면 원격을 연결하고 푸시한다. 원격 주소가 없으면 자동 푸시 설정은 `scripts/git-auto-push.ps1` 형태로 준비한다.
